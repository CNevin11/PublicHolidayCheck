# Public Holiday Decision Helper

This simple GitHub Pages app helps agents determine what to do during a public holiday week.

## How to deploy
1. Create a new GitHub repo named `ph-decision-helper`.
2. Upload `index.html` to the repo root.
3. Go to **Settings → Pages → Source → Deploy from a branch → main / (root)**.
4. Visit `<your-username>.github.io/ph-decision-helper` to use it.

No build step required — runs entirely client-side using React via CDN.
